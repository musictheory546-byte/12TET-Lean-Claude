# 12-TET with Lean using Clause

## Finding a set of axioms to ground 12-TET

What would be a (perhaps minimal) musically-justified set of axioms from which we could (perhaps neccessarily) derive the 12-TET tuning system. That is: a collection of axioms expressed in terms of a ring mod m, which constrain m to be exactly 12, with a sutable structure represting concepts such as a "tritone" interval, a pair of mutually-inverse "perfect" intervals, and the major/minor intervals to fill up the rest of the 12.

## Process
The initial candidate set of axioms were created by noodling with Claude, using Lean 4 to confirm at every step that Claude was producing a sound, valid argument.

This proof was then refined and where possible reduced, using Claude in a GAN-like mode to challenge it's own work.
